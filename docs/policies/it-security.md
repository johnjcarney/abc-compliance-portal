# IT Security & Acceptable Use

## 1. Password Standards
* Minimum length: **12 characters**.
* Must be changed every 90 days.
* **Prohibited:** Sharing passwords with colleagues or students.

## 2. Zalo & Chat Apps
!!! danger "Strict Prohibition"
    Sending files containing **Sensitive Personal Data** (e.g., Student IDs, Medical Reports) via Zalo, Viber, or Messenger is **strictly prohibited**. 

    Use the official School Email or Document Portal instead.

## 3. Device Security
* All laptops must have BitLocker encryption enabled.
* Screens must be locked (`Windows + L`) when leaving your desk.
