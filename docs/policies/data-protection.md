# Data Protection & Classification Policy

## 1. Data Classification Matrix
We classify data into two tiers as required by law.

| Classification | Definition | Examples | Security Requirement |
| :--- | :--- | :--- | :--- |
| **Basic Data** | General identification info. | Name, DOB, Personal Email, Citizenship. | Password Protection. |
| **Sensitive Data** | Data causing harm if leaked. | **Health records**, **Financial status**, **Biometrics**, **Religion**. | **Encrypted** & **Audit Logged**. |

## 2. Data Minimization Principle
!!! quote "The Golden Rule"
    "If you do not need it for a specific educational purpose, do not collect it."

## 3. Storage Rules
* **Physical:** All sensitive paper files must be in locked cabinets. Keys are restricted to authorized personnel.
* **Digital:** Sensitive data must NOT be stored on local desktops or USB drives. Use only the approved School Management System.
